학습 시리즈

SERIES 01 · 9개 모듈

openclaude로 배우는 에이전트 내부 원리

Claude Code 계열 에이전트 코딩 CLI를 분석 — 에이전트 루프, 툴 시스템, 컨텍스트 엔지니어링, 멀티모델 추상화, 서브에이전트, 확장성, 권한까지. 모든 코드 예시는 실제 소스의 파일:라인으로 검증.

 SERIES 02 · 9개 모듈

superpowers로 배우는 에이전트 방법론

"에이전트가 어떻게 일해야 좋은 코드를 만드는가" — 방법론 레이어 분석. Skill 해부학, 부트스트랩·자동 트리거, brainstorm→plan→execute 파이프라인, TDD, 서브에이전트 병렬화, 리뷰·검증 루프, 행동 형성 언어 기법까지.

이 노트의 원칙

왜 → 다이어그램 → 메커니즘 → 실제 코드 예시 → 직접 만든다면. 각 주제를 이 5단계로 풀어, "이 개념이 왜 필요하고, 어떻게 구현됐고, 내가 만든다면 어떻게 할지"를 설명할 수 있게 하는 것을 목표로 합니다. 코드 인용은 추측이 아니라 실제 소스를 대조해 적습니다.
