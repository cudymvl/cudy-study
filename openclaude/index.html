이 학습 자료에 대하여

이 사이트는 openclaude (Claude Code 계열 에이전트) 코드베이스를 분석한 학습 자료 세트입니다. 단순 사용법이 아니라 "어떤 핵심 개념들이 있고, 그것들이 실제로 어떻게 구현되었는가"를 깊게 파고듭니다.

각 모듈은 독립적으로 읽을 수 있지만, 0 → 1 → 2 → … → 8 순서로 읽으면 큰 그림에서 세부로 자연스럽게 진입합니다. 모듈 0(전체 조감도)과 모듈 1(한 턴의 흐름)을 먼저 읽어 전체 구조를 파악하는 것을 권장합니다.

학습 방법: 각 모듈은 (1) 왜 이 개념인가 → (2) 아키텍처 다이어그램 → (3) 핵심 메커니즘 해설 → (4) 실제 코드 예시 → (5) "직접 만든다면" 교훈 순서로 구성됩니다.
